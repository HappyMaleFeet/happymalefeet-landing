# HappyMaleFeet Landing Page

Coming soon landing page for HappyMaleFeet with Telegram and Throne links.

## Files
- `index.html` - Main landing page
- `HMF_Logo_transparent.png` - Logo image

## Deployment

### Option 1: Deploy to Namecheap via FTP
1. Log into Namecheap cPanel
2. Open File Manager
3. Navigate to `public_html`
4. Upload both files

### Option 2: Deploy via GitHub Pages
1. Push this repo to GitHub
2. Enable GitHub Pages in repo settings
3. Point your Namecheap domain to GitHub Pages

## Local Development
Simply open `index.html` in your browser to preview changes.
